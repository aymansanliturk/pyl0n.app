# libs/

Place the following files in this folder for fully offline operation:

| Filename | Download from |
|----------|--------------|
| `xlsx.full.min.js` | https://cdn.sheetjs.com/xlsx-0.20.3/package/dist/xlsx.full.min.js |
| `html2pdf.bundle.min.js` | https://cdnjs.cloudflare.com/ajax/libs/html2pdf.js/0.10.1/html2pdf.bundle.min.js |
| `html2canvas.min.js` | https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js |
| `chart.js` | https://cdn.jsdelivr.net/npm/chart.js/dist/chart.umd.min.js |
| `fonts.css` | Generated by `node scripts/download-libs.js` |

Also place the `fonts/` subfolder here (populated by the download script).

Or run once to download everything automatically:
```bash
node scripts/download-libs.js
```
