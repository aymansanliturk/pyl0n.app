# libs/

Place the following files in this folder for fully offline operation:

| Filename | Used by |
|----------|---------|
| `xlsx.full.min.js` | All tools with Excel import/export |
| `html2pdf.bundle.min.js` | PDF export in all tools |
| `html2canvas.min.js` | TimeCast / OrgCast PNG export |
| `chart.js` | CashFlow, W2W Report |
| `pptxgen.bundle.js` | BidPack PPTX export |
| `msal-browser.min.js` | Cloud sync / auth (BidPack, ForeCast) |
| `fonts.css` | Generated by download script |

Also place the `fonts/` subfolder here (populated by the download script).

Or run once to download everything automatically:
```bash
node scripts/download-libs.js
```
